# 🃏 德州扑克 AI 训练器

9 人桌德州扑克，单人训练 + 多人联机。

## 快速开始

```bash
# 安装依赖
cd server && npm install

# 启动服务器
cd .. && npm start

# 打开浏览器
# 单人训练: http://localhost:3000
# 联机对战: http://localhost:3000/client/
```

## 联机玩法

1. 一台电脑启动服务器（`npm start`）
2. 打开 `http://localhost:3000/client/` → 创建房间 → 获得房间码
3. 朋友（同一局域网）打开 `http://<你的IP>:3000/client/` → 输入房间码 → 加入
4. 房主点击「开始游戏」，AI 填满空位

## 游戏参数

| 参数 | 值 |
|------|-----|
| 起始筹码 | 20,000 |
| 初始盲注 | 100/200 |
| 盲注等级 | 100/200 → 200/400 → 300/600 → 400/800 → 500/1000 → 1000/2000 → 2000/4000 → 5000/10000 |
| 升盲条件 | 每 20 手牌 或 每 10 分钟，第 3 人起淘汰即升 |
| 加注选项 | +100% / +150% / +200% |
| 回合超时 | 60 秒（联机） |

## 双模式

| | 训练模式 | 竞技模式 |
|------|:--:|:--:|
| AI 性格标签 | ✅ 显示 | ❌ 隐藏 |
| 手牌亮明 | 每局结束全亮 | 仅摊牌亮牌 |
| 弃牌者手牌 | 亮 | 隐藏 |

## 命令

```bash
npm start          # 启动服务器
npm run restart    # 重启服务器（自动杀旧进程）
npm test           # 运行集成测试
```

## 技术栈

纯 HTML/CSS/JS 前端 + Node.js + Socket.IO 后端，shared/ 目录前后端共享代码。
